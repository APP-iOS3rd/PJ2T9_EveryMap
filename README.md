# 서비스 소개

### EveryMap, 목적지까지 더 즐겁고 편리한 여정을 안내합니다. 함께 떠나볼까요?
```
EveryMap은 다중 네비게이션 경로 비교 및 경로 최적화 서비스입니다.

EveryMap의 시작은
"네비게이션 앱들은 많고 같은 곳을 가더라도 각 앱마다 걸리는 시간과 경로가 다른데 뭐가 더 좋은거지?" 라는
질문에서 시작되었습니다.
```
